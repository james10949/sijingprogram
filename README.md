# Commands
  本機器人的前墜代碼為$符號
  help 在Discord查看指令列表
  clean <int> 清除指定數量訊息(須具有管理員權限)
  about 顯示關於頁面(本頁)
  ping 顯示延遲(單位ms)
  said 由機器人代發訊息
  picture 顯示隨機圖片(幾乎都是VOCALOID相關圖片)
	
	其他功能正在開發中!!!
